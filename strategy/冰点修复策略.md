# 冰点修复策略
## 核心逻辑：情绪冰点后次日情绪修复，套利卖出

---

## 策略参数（待回测优化）

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `ICE_POINT` | 25 | 冰点阈值 |
| `REPAIR_TARGET` | 40 | 修复目标（卖出参考） |
| `HOLD_DEADLINE` | 10:00 | 最晚持有时间（10点前必须卖） |
| `STOP_LOSS` | -5% | 止损线 |
| `MIN_LIANBAN` | 2 | 最小连板数（筛选标的） |

---

## 策略逻辑

```
【每日收盘后扫描】
IF 今日情绪 ≤ 25（冰点）
    → 标记为"冰点日"
    → 记录：冰点分、连板数据、涨停股
    
【次日开盘执行】
IF 昨日是冰点日
    → 计算竞价情绪（开盘分 - 昨日收盘分）
    → IF 情绪已修复（>40）OR 时间>10:00
        → 卖出全部
    → IF 情绪未修复 AND 时间<10:00
        → 继续持有观察
```

---

## 买入标的筛选

### 方案1：昨日涨停股（首板）
- 逻辑：首板股次日有溢价惯例
- 筛选：封单金额 > 5000万

### 方案2：二连板股
- 逻辑：资金明确接力，连板概率高
- 筛选：昨日二板 + 板块强势

### 方案3：百日新高股
- 逻辑：趋势强劲，延续性强
- 筛选：当日创新高 + 板块加持

---

## 卖出条件（优先级）

1. **情绪修复卖出**：次日开盘情绪 > 40
2. **时间强制卖出**：10:00前无论盈亏
3. **止损卖出**：持仓跌幅 > 5%
4. **尾盘卖出**：14:30仍未触发以上条件
