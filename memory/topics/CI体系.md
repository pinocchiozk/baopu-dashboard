# CI体系 - 产业情报追踪

*灵感来源：AGENTS.md 产业情报员角色 + RFC/PEP文档编号惯例*
*建立时间：2026-04-09*

---

## 概念

CI = **C**orporate **I**ntelligence（产业情报）

每个重大产业动态分配唯一编号，方便追踪、调用、避免重复。

---

## 编号规则

```
CI-{产业简称}-{序号}
```

| 示例 | 含义 |
|------|------|
| CI-PCB-001 | PCB产业第1条情报 |
| CI-液冷-001 | 液冷产业第1条情报 |
| CI-光储-001 | 光储出海第1条情报 |

---

## 评级制度

| 级别 | 触发条件 | 动作 |
|------|---------|------|
| 🔴 重大 | 政策转向/业绩大变/技术突破 | 立即推送飞书+建议操作 |
| 🟡 重要 | 产业链变化/数据更新/产能进度 | 次日推送+观察 |
| 🟢 一般 | 常规跟踪/财报跟踪 | 周报汇总 |

## 使用场景

1. **每日复盘**：读取CI档案，确认持仓标的是否有重大更新
2. **个股分析**：调用相关CI编号，避免重复收集信息
3. **周报输出**：汇总本周CI情报变化

## 📋 CI-XXX 创业板新高：算力硬件+锂电驱动（2026-04-14）

**市场背景：** 创业板阶段新高，算力硬件+锂电双驱动

### 驱动一：算力硬件

**"二力二光"框架（马博提炼）：**
- 出海算力 + 国产算力 齐涨
- 光通信（"二光"之一）

**光通信三层结构：**

| 层次 | 细分 | 近期表现 |
|------|------|---------|
| 第一层 | **光模块**（中际旭创/天孚/新易盛） | 持续大涨 |
| 第二层 | **CPO + OCS** | 滞涨整理 |
| 第三层 | **光芯片 + 硅光设备** | 滞涨整理 |

**关键事件：**
- Q1光模块业绩利好被机构放大（共识形成）
- 3月已预判光通信，战争扰动后共识扩散
- **DCI（数据中心互联）** 招标增加：谷歌/META/微软
  - DCI ≠ Scale-across，是数据中心间基础设施互联方案
  - 成本最大为相干光模块+光纤

**风险：** 午后光通信跳水引发创业板波动（高位分歧），但锂电板块接力+机构回补创业板ETF承接

### 驱动二：锂电板块

- 接力光通信成为第二动力
- 之前弱势时已提前跟踪

### 机构行为
- 大量机构回补创业板ETF（之前战争中减仓）
- 光通信已成创业板权重成分

### 操作策略
- 中线：继续看好光通信（含通信ETF）
- 短线：涨幅快→高位波动预期；未持仓不追高；有仓持盈或止盈
- OCS/CPO/光芯片滞涨正常，非不行了
