### 操作步骤（MuMu 模拟器 1440x2560）

**唯一路径：首页→市场情绪图标**

1. **启动开盘啦 App**
   ```bash
   adb -s 127.0.0.1:16384 shell monkey -p com.aiyu.kaipanla -c android.intent.category.LAUNCHER 1
   ```

2. **点击市场情绪图标**
   - 位置：首页顶部第一行第二个（带火焰🔥图标）
   - 坐标：`adb -s 127.0.0.1:16384 shell input tap 440 252`
   - 说明：坐标通过 uiautomator dump 获取，bounds=[299,167][582,338]，中心点 (440, 252)

3. **下滑查看完整数据**
   ```bash
   adb -s 127.0.0.1:16384 shell input swipe 500 1500 500 800
   ```

4. **截图获取数据**
   ```bash
   adb -s 127.0.0.1:16384 shell screencap -p > /tmp/mumu_sentiment.png
   ```

5. **返回**
   ```bash
   adb -s 127.0.0.1:16384 shell input keyevent 4
   ```
