## 六、核心结论

### ✅ 看多逻辑

1. **季度环比正增长：** 2025年Q1–Q3连续正增长（营收QoQ +28.7%→-6.6%，净利润QoQ +2.2%→+6.1%），Q4季节性回落不改向上趋势
2. **PEG极度低估：** 2026年预测PEG仅0.49（29.88%增速/14.6x PE），机构一致预期强劲
3. **储能爆发：** 2025Q1储能发货量同比+470%，Q4淡季压利润不改变长期趋势
4. **AIDC第二曲线：** 布局算力数据中心，打造第二增长极
5. **龙头溢价：** 全球光储逆变器龙头，海外收入占比持续提升

### ⚠️ 风险因素

1. **止损迫在眉睫：** 当前价¥123.32距止损价¥122.50仅0.66%，纪律优先
2. **Q4业绩季节性差：** Q4净利润仅15.8亿元（QoQ-61.9%），反映淡季特征
3. **PB 5.49x偏高：** 绝对估值不便宜，需高增长消化
4. **光伏政策风险：** 补贴退坡、装机量不及预期
5. **汇率风险：** 海外收入占比高，人民币升值压缩利润
