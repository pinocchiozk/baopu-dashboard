### 二、美国关税政策对光储出海影响

**最新关税税率（2026年1月起执行）：**

| 产品类别 | 综合关税率 |
|---------|-----------|
| 储能电池/系统 | **48.4%**（基础3.4% + 301条款25% + 双10%叠加） |
| 锂电材料 | **80%~84.8%** |
| 逆变器 | **81.5%**（基础2.5% + 45%附加 + 34%新增） |
| 光伏组件 | **70%**（2025年3月后） |

**关税政策要点：**
- 2025年3月4日生效：新增双10%关税叠加措施
- 2025年2月：特朗普签署行政令对华商品加征10%关税
- 2024年5月：锂电池关税从7.5%提至25%
- 2026年将进入关税全面落地期，中企有约1年缓冲期建设海外工厂

**对阳光电源影响分析：**
- 阳光电源美国市场占比约15-20%，并非最核心市场（欧洲为主）
- 48.4%储能关税削弱价格竞争力，但美国储能市场90%依赖进口（磷酸铁锂），本土产能严重不足
- **行业已在加速东南亚、中东产能布局规避关税**
- 2025-2026可能引发美国市场抢装效应（抢在关税全面落地前）
- 逆变器81.5%的税率对出口美国逆变器影响极大

**情报评级：🟡重要** — 关税压力持续，但阳光电源欧洲基本盘稳固，美国可通过海外工厂绕道

### 三、国内光伏政策变化

**🔴重大政策变动：2026年4月1日起全面取消光伏出口退税**

| 时间节点 | 政策内容 |
|---------|---------|
| 2024年11月 | 出口退税率从13%下调至9% |
| 2025年4月1日 | 退税率从9%降至**0**（249项光伏产品全面取消） |

**政策核心动因：**
- 终结"内卷外化"：2025年光伏全行业亏损超310亿元
- 组件均价跌至0.095美元/瓦，低于成本线
- 部分企业将退税折算为议价空间，财政资金变相补贴境外采购方

**对行业影响：**
- 每块210R组件出口利润减少46-51元
- 约30%低效产能在二季度面临加速出清
- 头部企业（隆基、天合、晶科）已纷纷上调价格：天合今年累计+8.5%，晶科高功率产品涨幅高达50%

**2026年国内光伏补贴走向：**
- 2025年作为国家补贴最后一年（0.3元/度）
- 2026年起全面取消保障性电价，强制平价上网
- 地方补贴和绿证交易成为主要支持方式

**情报评级：🔴重大** — 退税取消推高成本，加速行业产能出清，头部企业反而受益
