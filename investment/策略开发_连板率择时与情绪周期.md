## 策略一：连板率择时

### 1.1 核心理念

连板率 = 二板数 / 一板数

**逻辑：**
- 一板数量反映市场活跃度（有多少股尝试涨停）
- 二板数量反映接力意愿（有多少股能够连板）
- 连板率高 → 资金愿意接力 → 赚钱效应扩散 → 行情延续
- 连板率低 → 资金观望 → 赚钱效应萎缩 → 行情结束

### 1.2 指标定义

```
连板率 = 二板数量 / 一板数量 × 100%

辅助指标：
- 最高板天数：市场中最高连板是几板
- 三板率 = 三板数 / 二板数 × 100%
- 断板率 = 昨日连板今未涨停数 / 昨日连板数 × 100%
```

### 1.3 择时信号

| 连板率区间 | 市场状态 | 操作建议 |
|-----------|---------|---------|
| <10% | 🧊冰点 | 关注强势股，逆向布局 |
| 10%-25% | 📊偏低 | 谨慎做多，控制仓位 |
| 25%-40% | 📈正常 | 顺势而为，正常操作 |
| 40%-60% | 🔥偏热 | 赚钱效应扩散，可积极参与 |
| >60% | ⚠️过热 | 警惕高潮，注意撤退 |

## 策略二：情绪周期

### 2.1 核心理念

市场情绪存在周期性波动：
- 冰点 → 转暖 → 升温 → 高潮 → 退潮 → 冰点

通过情绪分数和赚钱效应的周期位置，指导仓位和标的选择。

### 2.2 情绪分数定义

```
情绪分数 = f(综合强度, 涨停家数, 跌停家数, 连板率, 量能变化)

简化公式：
情绪分数 = 50 + 0.3×(涨停数-30) - 0.3×(跌停数-20) + 0.2×(量能变化) + 0.2×(连板率-15)
```

### 2.3 情绪周期阶段

| 阶段 | 情绪分 | 特征 | 操作 |
|------|--------|------|------|
| 冰点 | 0-25 | 跌停多、涨停少、市场恐惧 | 逆向布局，观察主线 |
| 转暖 | 26-40 | 涨停增加、跌停减少 | 试仓，确认主线 |
| 升温 | 41-55 | 赚钱效应扩散、连板增加 | 积极参与，顺势而为 |
| 高潮 | 56-75 | 涨停潮、全民炒股 | 持有为主，逐步撤退 |
| 退潮 | 76-90 | 涨停减少、跌停增加 | 降低仓位 |
| 冰点 | >90 | 市场恐惧、流动性危机 | 清仓观望 |

### 2.4 情绪与大盘关系

**历史规律：**
- 情绪分数低点 → 大盘阶段性底部
- 情绪分数高点 → 大盘阶段性顶部

```
顶背离：大盘创新高，但情绪分数未能创新高 → 顶部信号
底背离：大盘创新低，但情绪分数未能创新低 → 底部信号
```

### 2.5 情绪择时策略

```python
def sentiment_timing(trade_date):
    data = get_daily_sentiment(trade_date)
    
    score = data['sentiment_score']
    limit_up = data['limit_up_count']
    limit_down = data['limit_down_count']
    lianban_rate = data['erban_rate']
    
    # 多维度确认
    if score <= 25 and limit_down > 30:
        return "冰点", "重仓买点", "逆向布局"
    elif score <= 25 and limit_down <= 30:
        return "偏冷", "轻仓观察", "等待确认"
    elif score >= 75 and limit_up > 60:
        return "过热", "减仓", "警惕顶部"
    elif score >= 75 and limit_down > 20:
        return "退潮", "清仓", "避险"
    else:
        return "正常", "标准仓位", "顺势而为"
```

### 2.6 情绪周期交易模板

```
【盘前】
1. 查看情绪分数 + 连板率
2. 判断市场所处周期阶段
3. 制定当日操作策略

【盘中】
1. 每30分钟监控情绪变化
2. 警惕情绪急剧变化（冰点→过热）
3. 根据信号调整仓位

【盘后】
1. 记录当日情绪分数
2. 分析连板率变化趋势
3. 预判次日市场情绪
```

```python
def backtest_strategy(start_date, end_date):
    """
    择时策略回测
    """
    results = []
    
    for date in date_range(start_date, end_date):
        # 获取当日数据
        market = get_market(date)
        lianban = get_lianban(date)
        sentiment = get_sentiment(date)
        
        # 计算信号
        signal = get_signal(lianban, sentiment)
        
        # 记录
        results.append({
            'date': date,
            'signal': signal,
            'market_return': market['return'],  # 大盘收益率
            'position': signal['position']
        })
    
    # 计算绩效
    total_return = sum(r['market_return'] * r['position'] for r in results)
    max_drawdown = calculate_max_drawdown(results)
    win_rate = calculate_win_rate(results)
    
    return {
        'total_return': total_return,
        'max_drawdown': max_drawdown,
        'win_rate': win_rate,
        'results': results
    }
```
