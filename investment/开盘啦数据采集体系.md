## 三、盘中采集（9:15-15:00）

### 3.1 市场情绪（入口: 350, 220）

**页面结构：**
```
市场情绪首页
├── 上段：指数+涨跌家数
├── 中段：涨停表现区域
│   ├── 一板(32) → 点击(?,?)进入详情
│   ├── 二板(3)  → 点击(343,1146)进入详情
│   ├── 三板(0)
│   ├── 四板(0)
│   └── 更高(1)
└── 下段：大幅回撤个股
```

**采集指标：**
| 指标 | 字段 | 投研意义 |
|------|------|---------|
| 综合强度 | 0-100分数 | 大盘温度 |
| 实际涨停 | 数量 | 赚钱效应 |
| 实际跌停 | 数量 | 亏钱效应 |
| 预测量能 | 金额 | 资金活跃度 |

**连板梯队（点击数字进入详情）：**
| 梯队 | 数量 | 连板率 | 投研意义 |
|------|------|--------|---------|
| 一板 | 32 | - | 首板赚钱效应 |
| 二板 | 3 | 14% | 连板接力情绪 |
| 三板 | 0 | 0% | 高位接力意愿 |
| 四板 | 0 | 0% | 龙头高度 |
| 更高 | 1 | 25% | 市场总龙头 |

**连板详情页字段：**
- 股票名称、代码
- 涨停时间
- 涨停原因（板块）
- 封单金额
- 备注（如"9天7板"）

### 3.2 复盘啦（入口: 440, 220）

**核心价值：** 当日异动追踪、热点确认

**采集内容：**
- 异动个股列表
- 异动原因分类
- 资金流向

### 3.3 百日新高（入口: 300, 160）

**页面位置：** 首页第二页第一行第二个

**核心价值：** 筛选强势股

**采集内容：**
- 百日新高个股列表
- 板块分布统计
- 成交额、换手率

### 3.4 行情-板块（入口: 270, 1520）

**入口坐标：** 底部导航栏第二个

**页面结构：**
```
行情-板块
├── 顶部：沪指 + 量能预测 + 涨跌家数
├── 尾盘抢筹提示
└── 板块列表（按强度排序）
    ├── 板块名称
    ├── 强度指数
    ├── 主力净额
    └── 机构增仓
```

**采集指标：**
| 指标 | 字段 | 投研意义 |
|------|------|---------|
| 沪指 | 点位、涨跌幅 | 大盘走势 |
| 预测量能 | 金额、环比 | 资金活跃度 |
| 涨跌家数 | 上涨/下跌 | 市场温度 |
| 板块强度 | 强度值 | 热点方向 |
| 主力净额 | 金额 | 主力动向 |
| 机构增仓 | 金额 | 机构偏好 |

**板块列表字段：**
- 板块名称（代码）
- 强度
- 主力净额（当日）
- 第四季度机构增仓

### 3.5 打板（入口: 336, 130）

**入口方式：** 行情页面 → 第三个tab

**页面结构：**
```
打板
├── 顶部统计：涨停/跌停/封板率
├── 子标签：竞价/即将涨停/风向标/涨停
└── 个股列表
    ├── 股票名称
    ├── 涨停委买额
    ├── 涨幅
    └── 所属板块
```

**子标签说明：**
| 标签 | 内容 | 投研价值 |
|------|------|---------|
| 竞价 | 竞价阶段异动 | 开盘方向 |
| 即将涨停 | 接近涨停个股 | 潜在标的 |
| 风向标 | 市场情绪风向 | 龙头判断 |
| 涨停 | 当日涨停明细 | 赚钱效应 |

**涨停个股字段：**
- 股票名称、代码
- 涨停委买额
- 涨幅
- 所属板块
- 游资标签
- 融资标签

```

四、强势个股追踪
   ├── 百日新高个股
   ├── 涨停个股分类
   └── 游资参与标的

五、明日展望
   ├── 消息面影响
   ├── 外盘指引
   └── 重点关注标的
```

## 五、数据采集流程

### 5.1 盘中采集频率

| 数据源 | 采集频率 | 预警阈值 |
|--------|---------|---------|
| 市场情绪 | 每30分钟 | 情绪分数≤25或≥75 |
| 行情-板块 | 每30分钟 | 板块强度突变 |
| 打板 | 每15分钟 | 涨停数异常 |

### 5.2 采集脚本模板

```bash
# 采集流程
1. 进入市场情绪 → 截图 → OCR识别情绪分数
2. 进入涨停表现 → 点击二板/三板/更高 → 截图 → OCR识别连板数据
3. 进入行情-板块 → 截图 → OCR识别板块排名
4. 进入打板 → 截图 → OCR识别涨停个股
```

## 七、预警规则

### 7.1 情绪预警

| 情绪分数 | 状态 | 操作建议 |
|---------|------|---------|
| ≤25分 | 🧊冰点 | 买点区域，关注强势股 |
| 26-74分 | 📊正常 | 持仓观察 |
| ≥75分 | 🔥过热 | 风险区域，谨慎追高 |

### 7.2 连板预警

| 指标 | 异常阈值 | 信号 |
|------|---------|------|
| 最高板 | 突然下降 | 龙头股断板 |
| 跌停数 | >30家 | 亏钱效应放大 |
| 破板率 | >40% | 封板资金不足 |
