## 模块说明

### 01-投资哲学与方法论

**定位：** 投资体系的"宪法"

**核心内容：**
- 核心收益来源（企业成长、估值波动、套利、趋势、逆向）
- 投资原则（价值第一、逆向思维、能力圈、安全边际、长期主义）
- 能力圈与聚焦（AI 科技赛道）
- 抱朴哲学（见素抱朴、少私寡欲、绝学无忧）
- 决策检验清单

**文档：** `00-总纲.md`
