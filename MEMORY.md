## 核心原则（2026-03-29 写入）

### 📊 投资研究数据原则

**基础数据必须来自真实来源，严禁使用假数据**

这是 z 明确强调的投资研究生命线，违背等于自杀。

| 原则 | 具体要求 |
|------|----------|
| 数据来源 | 必须来自真实来源（QVeris、财报、权威数据源） |
| 严禁行为 | 严禁假设数据、编造数据、估算数据 |
| 工具使用 | 有 QVeris 等工具必须使用 |
| 无法获取 | 明确告知用户，不提供分析 |
| 数据标注 | 所有数据标注来源，可验证 |

### 违规案例（2026-03-29）

**华工科技研究错误：**
- 错误：假设股价 35 元（实际 109.5 元）
- 后果：PEG 从 1.24 错算成 0.26
- 结论：从"合理偏高"错判为"严重低估"
- 教训：完全相反的结论，可能误导投资决策
